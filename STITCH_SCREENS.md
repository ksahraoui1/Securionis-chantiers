## Design System — Blueprint Precision

| Token | Valeur | Usage |
|-------|--------|-------|
| `primary_container` | `#131B2E` | Top bar, CTA bouton |
| `secondary` | `#006E2D` | Conforme, succès |
| `on_tertiary_container` | `#F63A35` | NC, alertes, erreurs |
| `surface` | `#F9F9FF` | Fond général |
| `surface_container_lowest` | `#FFFFFF` | Cartes actives |
| `surface_container_low` | `#F0F3FF` | Sections |
| Font | Inter | Tous les niveaux |
| Radius cards | `xl` (1.5rem) | Toutes les cartes |
| Shadow | `0px 12px 32px rgba(21,28,39,0.06)` | FAB, cartes tappables |
